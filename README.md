## 📍 위치기반 근처 병원 대기 현황 조회, 비교 및 원격 예약 / 근처 약국 위치·대기 시간 조회 플랫폼

<위치 기반 주변 병원 검색, 대기 현황 조회, 비교 및 원격 예약 서비스> 

본 서비스는 사용자의 현재 위치를 기반으로 주변 병원을 조회하고,

환자의 주호소에 해당하는 진료과목에 실시간 환자 대기 수 확인 후 대기열이 적은 병원으로 예약 후 진료,

나아가 진료 기록 및 처방 등 관련 의료 정보들을 체계적으로 관리할 수 있게 도와주는 의료 편의 플랫폼입니다.

<근처 약국 위치·대기 시간 조회 플랫폼>

진료 이후 근처 약국에 대기 시간을 조회하고,

대기 시간이 가장 짧은 약국을 이용할 수 있도록 도와준다.

## 📍 프로젝트 소개

이 프로젝트는 환자가 병원에 방문하기 전에 해당 병원에 대기 인원이 얼마나 있는지를 조회해서 비교하고
나아가 원격으로 대기를 등록해 진료를 예약할 수 있도록 해
병원 방문 시 발생하는 무한 대기 현상을 줄이고 의료서비스 이용의 효율성을 극대화 하는 것을 목표로 합니다.

 **위치 기반 서비스(LBS)**와 실시간 데이터 처리를 활용하여

프론트엔드와 백엔드 간의 데이터 통신을 통해 병원 대기 정보를 제공하고,

사용자별 병원 방문 이력을 안전하게 관리할 수 있도록 설계되었습니다.

🏥 주요 기능

위치 기반 병원 검색

사용자 현재 위치 기준 주변 병원 자동 탐색

실시간 대기열 확인

병원별 대기 인원 및 예상 대기 시간 제공

병원 방문 이력 관리

개인별 진료 이력 및 방문 기록 저장·조회

병원 비교 기능

거리, 대기시간, 진료과목 기준 비교

🎯 기대 효과

불필요한 병원 대기 시간 감소

사용자 중심의 병원 선택 가능

개인 의료 이력 관리의 편의성 향상

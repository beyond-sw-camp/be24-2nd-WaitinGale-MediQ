## 📍 위치기반 근처 병원 대기 현황 조회, 비교 및 원격 예약 / 근처 약국 위치·대기 시간 조회 플랫폼

<위치 기반 주변 병원 검색, 대기열 확인, 내 병원이력관리 서비스> 

본 서비스는 사용자의 현재 위치를 기반으로 주변 병원을 조회하고,

실시간 환자 대기 수 확인 후 대기열이 적은 병원 , 개인 병원 방문 이력을 체계적으로

관리할 수 있는 의료 편의 플랫폼입니다.

## 📍 프로젝트 소개

이 프로젝트는 **위치 기반 서비스(LBS)**와 실시간 데이터 처리를 활용하여

의료 이용의 비효율성을 해결하는 것을 목표로 합니다.

프론트엔드와 백엔드 간의 데이터 통신을 통해 병원 대기 정보를 제공하고,

사용자별 병원 방문 이력을 안전하게 관리할 수 있도록 설계되었습니다.

🏥 주요 기능

위치 기반 병원 검색

사용자 현재 위치 기준 주변 병원 자동 탐색

실시간 대기열 확인

병원별 대기 인원 및 예상 대기 시간 제공

병원 방문 이력 관리

개인별 진료 이력 및 방문 기록 저장·조회

병원 비교 기능

거리, 대기시간, 진료과목 기준 비교

🎯 기대 효과

불필요한 병원 대기 시간 감소

사용자 중심의 병원 선택 가능

개인 의료 이력 관리의 편의성 향상
